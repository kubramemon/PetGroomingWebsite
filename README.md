# Pet Grooming Website

Pet Grooming Website is a simple and user-friendly platform that helps pet owners book grooming services, explore pet care tips, and keep their pets clean, healthy, and happy.


## Features

* View and book pet grooming services
* Categories for bathing, trimming, and styling
* Information and tips on pet care and hygiene
* Option for online booking (future feature)
* Contact form for customer inquiries
* Responsive design that works on both mobile and desktop



## Tech Stack

Frontend: HTML, CSS, JavaScript
Backend: PHP, MySQL
Tools: Git, GitHub, VS Code



## Getting Started

1. Clone the repository

   git clone https://github.com/yourusername/GroomingWebsite.git
   

2. Open the project folder

   cd GroomingWebsite

3. Open the file named index.html in your browser to view the website.
